This is collection of netsnmp-based ruby scripts, that I wrote when was working as network engineer in Corbina Telecom/Beeline.
allusers.rb - D-Link 3526/3028 cable tester script.
descr.rb - show switch system description
fwver.rb - show switch firmware version
invent.rb - take file line by line and show every switch ip mac-address
location.rb - show switch location
m.rb - show some switch state (port status and so on)
name.rb - show system name
ospf*.rb - group of scripts that build network map for d-link switches (36xx) based on ospf info
saveconfig.rb - save D-Link l2 switch config
syscnt.rb - show switch system contact
vlans.rb - show L2-switch vlans
